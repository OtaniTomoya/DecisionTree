# 決定木
### Fortranでの実装

## 構成
- modules : 決定木の本体
- main.f90 : 決定木の実行
- Makefile : 実行ファイルの作成
- makeMnistData.f90 : データの変形


## 実行方法
1. mnist.csvを用意
2. クローンしたディレクトリにmnist.csvを追加
3. 以下のコマンドでdatファイルを作成
```
$ gfortran makeMnistData.f90
$ ./a.out
```
4. 以下のコマンドで実行ファイルを作成
```
$ make
```
5. 決定木を実行
```
./DecisionTree
```
